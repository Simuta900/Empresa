Red 5G (Conferencia Platzi)

La red 5G es la evolución de las redes inalámbricas 4.5G (LTE), en la que además de conseguir una menor latencia (tiempo de respuesta al enviar y recibir paquetes de datos), se aumenta la capacidad de comunicación simultánea entre dispositivos, así como la velocidad de transmisión, llegando a velocidades teóricas en un entorno ideal de hasta 10Gbps (100 veces mayor que la tecnología actual). De igual forma, con esta tecnología, abrimos la posibilidad de trabajar con dispositivos IoT (el famoso “Internet de las cosas”). Gracias a él, proveemos de alta capacidad de transmisión de datos a un sinfín de dispositivos con los que convivimos a diario (desde nuestra nevera a una simple botella de agua) y que nos permitirán recibir información a tiempo real de todo lo necesario para interactuar de forma remota en este mundo globalizado en el que vivimos.

Glosario con términos informáticos

Programación: Proceso en cual una persona crea o desarrolla un programa o aplicación por medio de instrucciones, las cuales están escritas en un cierto lenguaje (HTML, CSS, JavaScript, etc.), que es traducido a un lenguaje de máquina, y es interpretado y ejecutado por el procesador; y estas instrucciones las sigue el ordenador para ejecutar dicho programa o aplicación.

DOM(Document Object Model): Forma en la que internamente el navegador organiza todo el HTML dentro de una estructura de árbol.

HTML(Lenguaje de Marcas de Híper Texto): Lenguaje donde se define la información o el contenido del documento.

CSS(Lenguaje de estilos de cascada): Lenguaje donde se especifica el diseño del documento; maneja todo lo relacionado con la parte visual.

JavaScript: Lenguaje que hace que todo sea interactivo y que nos permite crear sitios web, además de ser el lenguaje interpretado por el navegador.

Etiqueta: Marca con clase que delimita una región del código, la cuál de instrucciones necesarias o específicas al navegador para presentar la página en pantalla.

Variable: Elemento empleado para almacenar y hacer referencia a otro valor.

Condicional: Código ejecutado, siempre y cuando una circunstancia sea cierta.

Función: Herramienta que permite escribir código que se va a reusar múltiples ocasiones.

Evento: Funciones que suceden cuando ocurre algo, causado por el usuario.

Arreglo: Conjunto de datos ordenados por posiciones, asociados en una sola variable; puede ser de cualquier tipo (texto, luego números, después objetos, etc.). Se puede acceder a estos datos de manera independiente, o si están agrupados.

Objeto: Envoltura para código que hacen referencia a cierta acción de un programa (es más común en un navegador).

Clase: Define las características del objeto (altura, ancho, color, etc.).

Parámetro: Información que una función necesita para lograr algo.

String: Cadena de texto o texto libre; mensaje que no es una función, variable, entre otros elementos en el código.
